<h1>Dishbook Application</h1>
<h2>Project Description</h2>

DishBook is a mobile recipe app created to help users keep 
all of their recipes organized in one place. A lot of people save recipes on paper, 
in notes apps, or through screenshots, which can get messy and hard to manage over time. 
DishBook solves that problem by giving users one simple app where they can store and manage 
everything.  

With DishBook, users can create and save recipes with important details such as 
the recipe name, description, cook time, servings, ingredients, directions, notes, 
and images. This makes it easy to quickly pull up recipes while cooking or planning meals.  

The app also includes a search feature that allows users to find recipes by name, 
along with a favorites section where users can save their top recipes for faster 
access. The goal of the project was to make recipe storage simple, clean, and 
useful for everyday users.  

<h2>Main Features</h2>

  - Add new recipes with custom details  
  - Edit existing recipes anytime  
  - Delete recipes that are no longer needed  
  - Upload images for recipes  
  - Search recipes by keyword or name  
  - Favorite and unfavorite recipes  
  - Store recipe data locally on the device  
  - Clean and simple mobile layout for easy navigation
 
<h2>How to Run This Project</h2>

1. Open the DishBook solution in Visual Studio, or any other IDE.

2. Wait for all NuGet packages to restore automatically.

3. Build the solution to make sure there are no errors.

4. Select your preferred device:
  - Android Emulator
  - Physical Android Device
  - Windows Machine (if supported)
    
5. Press **Start** or **Run** to launch the application.

6. Once the app opens, begin adding and managing recipes.

<h2>Technologies Used</h2>

  - .NET MAUI – used to build the cross-platform mobile application
  - C# – main programming language used for app logic
  - XAML – used to design the user interface
  - SQLite – used for local data storage
  - Visual Studio – development environment used to build and run the project  

<h2>NuGet Packages Used</h2>

  - sqlite-net-pcl – used to connect the app to a local
    SQLite database and store recipe information on the device.

<h2>App Structure</h2>

The app includes multiple pages for different actions:

  - **Home Page** - shows saved recipes in a list/card format
  - **Add/Edit Page** - allows users to create or update recipes
  - **Recipe Detail Page** - shows full recipe information
  - **Favorites Page** - displays recipes marked as favorites

This structure helps keep navigation simple and easy to understand.

<h2>Why This Project Was Made</h2>
This project was created to solve a common everyday problem: keeping 
recipes organized and easy to access. Many people lose recipes or forget
where they saved them. DishBook gives users one reliable place to store meals
they enjoy and use often.
